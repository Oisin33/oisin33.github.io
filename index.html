DarkScan

A detector for manipulative “dark pattern” UI, confirmshaming, fake urgency, sneaky opt-outs, that learns to catch patterns it has never seen, not just match a keyword list.

Live demo
  • A tiered ML cascade: heuristics → TF-IDF classifier → embeddings → LLM, escalating only the cases the cheap tiers can't settle (cost-aware by design).
  • Generalises to reworded, unseen phrasings via character n-grams, and improves from user feedback through a gated distillation loop with model-collapse guardrails.
  • The real trained model runs in your browser, exported from scikit-learn and reimplemented in JS, verified identical to the Python original.
